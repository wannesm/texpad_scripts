# texpad_scripts
Helper scripts for [Texpad](https://www.texpad.com).

## Scripts

### texpad

(**Only for macos**)

Launch the Texpad application from the command line. Optionally with a file
path as an argument:

```
$ texpad discriminative_bias_for_psdd.tex
```
